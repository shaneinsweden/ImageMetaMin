To test this solution do these actions

1. If you have already run the solution (so that some files have metatdata) delete the files
   from the images2 folder and copy the original images (without metadata) into the images2 folder

2. Edit the file ImageMetaTests.cs so that the paths to the imagefolder (images2) and the metadatafile Images2-s1.csv 

3. Run in debug mode the unit test UpdateCatalog() in the ImageMetaTests.cs file

When you run it you can see the memory increasing over time
